# Quick Start Guide - New Features

## 🚀 What's New?

Your RIT IPR Portal now has a complete dashboard system with file upload and OCR!

## 📋 Quick Overview

### 1. Dashboard (Home Screen)
After login, you'll see:
- Your patent statistics
- Recent patents
- Quick action buttons

### 2. Sidebar Menu
Navigate easily with the sidebar:
- 🏠 **Dashboard** - Home
- 📝 **New Patent** - Start application
- 📚 **History** - View all patents
- 👤 **Account** - Your profile
- 🚪 **Logout** - Sign out

### 3. File Upload with OCR
Upload documents instead of typing:
- Click the upload zone in Step 1
- Select an image (PNG/JPG)
- Wait for OCR to extract text
- Review and edit as needed

### 4. Patent History
All your patents are automatically saved:
- View past applications
- See scores and recommendations
- Track your progress

### 5. Account Page
Manage your profile:
- View your information
- Check API key status
- See your statistics

## 🎯 How to Use

### Starting a New Patent

**Option 1: Type Manually**
1. Click "New Patent"
2. Fill in the form fields
3. Continue to analysis

**Option 2: Upload Document**
1. Click "New Patent"
2. Click the upload zone
3. Select your document image
4. Wait for OCR processing
5. Review extracted text
6. Edit if needed
7. Continue to analysis

**Option 3: Both (Recommended)**
1. Upload document for quick start
2. Edit and refine the extracted text
3. Add more details manually
4. Continue to analysis

### Viewing Your History
1. Click "History" in sidebar
2. See all your patents
3. Review scores and recommendations
4. Click "New Patent" to start another

### Checking Your Account
1. Click "Account" in sidebar
2. View your profile
3. Check API key status
4. See your statistics

## 💡 Tips

### For Best OCR Results
- Use clear, high-resolution images
- Ensure good lighting
- Avoid handwritten text (OCR works best with printed text)
- Check extracted text for accuracy

### Navigation
- Use the sidebar for quick navigation
- Click the ☰ button to toggle sidebar
- Sidebar remembers your preference

### History
- Patents are saved automatically
- Last 20 patents are kept
- History persists across sessions
- Stored securely in your browser

## 🔧 Troubleshooting

### OCR Not Working?
- Check file format (PNG, JPG, JPEG only)
- Ensure file size is under 10MB
- Try a clearer image
- Refresh the page and try again

### Sidebar Not Showing?
- Click the ☰ button in the navbar
- Check if you're on a logged-in page
- Refresh the page

### History Not Saving?
- Make sure you proceed to "Generate Forms"
- Check browser cookies are enabled
- Don't use incognito/private mode

## 📱 Mobile Users

- Sidebar collapses automatically on mobile
- Use ☰ button to open menu
- All features work on mobile
- Upload works with camera

Your RIT IPR Portal is now more powerful and easier to use. Start filing patents faster with the new dashboard and file upload features!
