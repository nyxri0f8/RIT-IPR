## Table Specifications

### Column Widths (in mm)
- S.No columns: 15mm
- Name columns: 45mm
- Address columns: 85-120mm
- Code columns: 35mm
- Description columns: 120mm
- Field labels: 50mm

### Font Sizes
- Headers: 14pt bold
- Table headers: 10pt bold
- Table body: 9pt
- Body text: 11pt
- Footer: 9pt

### Color Scheme
- Primary blue: RGB(43, 95, 143)
- Text: RGB(0, 0, 0)
- Footer text: RGB(100, 100, 100)
- Borders: RGB(0, 0, 0) or RGB(200, 200, 200)
